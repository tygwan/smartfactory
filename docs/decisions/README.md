# Decisions (D-records)

Structural decision records for SmartFactory. Each new architectural or
structural choice — "X vs Y", framework adoption, scope boundary, irreversible
trade-off — gets its own `D-{NNN}-{slug}.md` file in this directory.

Reference:
- Standard: [aegis/standards/02-decisions.md](https://github.com/tygwan/aegis/blob/main/standards/02-decisions.md)
- Template: [aegis/templates/D-record.template.md](https://github.com/tygwan/aegis/blob/main/templates/D-record.template.md)

D-001 opens when the first such choice is made.
