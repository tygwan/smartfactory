# Plans (P-records)

Plan records for SmartFactory. Before non-trivial multi-step work (more than
one day, or more than three sub-tasks), a `P-{NNN}-{slug}.md` file captures
the approach, scope, sub-tasks, dependencies, and risks.

Reference:
- Standard: [aegis/standards/03-plans.md](https://github.com/tygwan/aegis/blob/main/standards/03-plans.md)
- Template: [aegis/templates/Plan-record.template.md](https://github.com/tygwan/aegis/blob/main/templates/Plan-record.template.md)

P-001 opens before the first qualifying multi-step task.
