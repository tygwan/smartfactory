# Verifications (V-records)

Verification records for SmartFactory. At the end of each meaningful work
unit (typically a closed plan, a feature ship, or a milestone gate), a
`V-{NNN}-{slug}.md` file records what was verified, how, and the outcome.

Reference:
- Standard: [aegis/standards/04-verification.md](https://github.com/tygwan/aegis/blob/main/standards/04-verification.md)
- Template: [aegis/templates/V-record.template.md](https://github.com/tygwan/aegis/blob/main/templates/V-record.template.md)

V-001 opens at the first meaningful work completion.
